# ZapRemote — Product Blueprint

**Status:** Active · **Priority:** Soccer reliability · **Price:** **$1.99/mo** flat

> Pick a **soccer** game → sync Match Clock → tap a highlight → rewind → watch → return.

## Business focus

| Goal | Target |
|------|--------|
| Sport | **Soccer only** (now) |
| Price | **$1.99/month** (no Pro tier) |
| Users | **10,000 paying** ≈ **$20k/mo** gross |
| Next sport | **NFL** only after soccer loop is solid |

Do **not** chase basketball, multi-sport sprawl, or “billion dollar” fantasy until soccer + 10k is real.

## Data

| Source | Use? |
|--------|------|
| **ESPN** | Primary — plays, clock, search |
| football-data.org | Optional soccer fixture helper only |

## User flow (golden path)

1. Connect LG TV + YouTube TV  
2. Pick **soccer** game  
3. Sync Match Clock  
4. Tap highlight (goals / cards / free kicks / corners)  
5. Rewind → watch → auto-return  

## Reliability (Phase 0)

- Manual **Ad** / highlight tap only (no auto commercial skip yet)  
- One highlight per break  
- Steady skip clicks + click-ledger return  
- Verify on real game nights before feature expansion  

## Later: NFL

Same loop (timeouts + TDs / turnovers). Do **not** start until soccer return + sync feel trustworthy.
